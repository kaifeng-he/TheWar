# The War (战争)

一个基于 React 19 和 HTML5 Canvas 构建的 2D PVE 即时战略（RTS）游戏。

![Version](https://img.shields.io/badge/version-0.1.0-blue)
![React](https://img.shields.io/badge/React-19-61DAFB)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6)

## 🎮 游戏简介

"The War" 旨在重现经典 RTS 游戏（如《红色警戒》或《帝国时代》）的竞技体验。玩家将选择不同的派系，在广阔的地图上采集资源、建造基地、训练部队，并最终击败敌方势力。

### 核心特性

- **多派系体验**：
  - **人类 (Human)**：各项属性平衡，适合新手。
  - **机器人 (Robot)**：高科技部队，造价昂贵但战力惊人。
  - **异形 (Alien)**：廉价的蜂群战术，依靠数量压制。
- **资源管理**：通过采集金矿和矿产来支撑战争扩张。
- **基地建设**：建造指挥中心、兵营、精炼厂和防御塔。
- **宏大战场**：支持 4000x4000 的超大可滚动地图，体验大规模兵团作战。
- **自定义对局**：支持调整敌人难度、初始资源和派系选择。

## 🛠 技术架构

本项目采用纯前端技术栈实现，不依赖外部游戏引擎，追求高性能与高度可定制性。

- **渲染引擎**：基于 HTML5 Canvas 的自定义渲染系统 (`GameCanvas.tsx`)。
- **游戏逻辑**：核心逻辑高度抽象于 `useGameLoop` Hook 中，处理状态更新、寻路、战斗及 AI。
- **状态管理**：结合 React `useState`（用于 UI 状态）和 `useRef`（用于高性能游戏循环数据）。
- **性能优化**：针对大规模单位同时在场进行了优化，确保流畅的 60fps 体验。

## 🚀 快速开始

### 运行环境
- Node.js (建议 v18+)
- npm 或 yarn

### 安装与启动
1. 进入项目目录：
   ```bash
   cd the-war
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 访问：`http://localhost:5173`

## ⌨️ 操作指南

- **地图滚动**：鼠标移动至屏幕边缘或使用 `WASD` / 方向键。
- **单位选择**：左键点击或框选。
- **移动/攻击**：右键点击目标点或敌方单位。
